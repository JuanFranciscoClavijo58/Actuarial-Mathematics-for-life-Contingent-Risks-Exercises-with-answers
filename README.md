# Actuarial-Mathematics-for-life-Contingent-Risks-Exercises-with-answers


These are the solutions to the exercises in the book *Actuarial Mathematics for Contingent Life Risks*, intended for actuaries.

The goal is to understand the process that leads to the solution. To understand life insurance—its structure, its actuarial notation—and how we can apply this knowledge in the workplace. We welcome any feedback that helps improve each exercise.
